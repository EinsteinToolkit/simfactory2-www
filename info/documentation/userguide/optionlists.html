Option Lists¶

An option list is a file containing a set of variable definitions used to configure a Cactus build on a particular machine. For example, it might contain the path to the C compiler, the desired optimisation settings, and the location of the HDF5 library on that machine.

SimFactory allows Cactus option lists to contain variable definitions which it substitutes. These definitions are refered to in the optionlist using the @DEF@ syntax, where DEF is the name of the definition. SimFactory converts the templated option list into a real option list by expanding these quantities.
