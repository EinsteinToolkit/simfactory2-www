create¶

create a simulation

Usage¶

sim create [options] simname [ parfile | --parfile parfile ]

Create a simulation named simname using a specific parameter file.

sim create [options] simname –testsuite [–select-tests testspec ]

Create a simulation named simname which runs the Cactus test suite. See the documentation for selecting tests.

Options¶

--parfile <file>¶

parameter file for executing Cactus simulation

--configuration <file>¶

configuration to use for Cactus simulation

--config <file>¶

see –configuration

--testsuite¶

Run Cactus testsuite

--select-tests all|<test>.par|<arrangement>|<arrangement>/<thorn>¶

Select which tests to run
